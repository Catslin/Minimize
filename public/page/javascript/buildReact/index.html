The createElement method in React and the naming of {type, props} are React's internal conventions and are used to represent the structure of React elements. Parameters to the createElement method include the element's type (e.g. "h1") and attribute object (e.g. { title: "foo" }), as well as optional child elements. This method is closer to the expression of JSX syntax. The naming method of {type, props} represents the type and attributes of the element more intuitively. type represents the type of element, such as "h1", and props is an object containing element attributes, including attributes such as title and children (or array attributes)..
